# 🔍 숨은 단어 찾기 생성기 (Word Search Generator)

아이들을 위한 교육용 단어 찾기 학습지를 자동으로 생성하고 PDF로 다운로드할 수 있는 웹 애플리케이션입니다. 사용자가 입력한 단어 목록을 바탕으로 매번 새로운 퍼즐을 생성하며, 정답지까지 함께 제공합니다.

## ✨ 주요 기능

- **24x24 대형 그리드**: 다양한 단어를 빽빽하게 배치할 수 있는 넉넉한 크기의 격자를 제공합니다.
- **18개 단어 고정 시스템**: 학습 효과를 극대화하기 위해 한 페이지에 가장 적합한 18개 단어로 입력을 고정하여 깔끔한 레이아웃을 제공합니다.
- **방대한 카테고리 라이브러리**: 직업, 동물, 과일, 나라, 스포츠 등 10개 이상의 카테고리와 총 2,000개 이상의 단어 데이터를 내장하고 있습니다.
- **스마트 단어 추출 및 섞기**:
  - **카테고리 추출**: 선택한 카테고리 내에서 무작위로 18개 단어를 불러옵니다.
  - **🎲 무작위(전체) 모드**: 카테고리 제한 없이 전체 데이터베이스에서 단어를 혼합하여 추출합니다.
  - **🔄 단어 섞기**: 클릭 한 번으로 입력창의 단어들을 새로운 조합으로 즉시 변경합니다.
- **다국어 모드 지원**: 
  - **영어 모드**: 알파벳 대문자로 구성된 퍼즐을 생성합니다.
  - **한국어 모드**: 한글 음절로 구성된 퍼즐을 생성하며, 빈칸 역시 무작위 한글로 채워집니다.
- **세분화된 난이도 (1~100)**: 1부터 100까지의 수치에 따라 가로, 세로, 대각선, 역방향 배치를 동적으로 조절하여 정교한 난이도 설정이 가능합니다.
- **자동 PDF 생성**: A4 가로(Landscape) 모드에 최적화된 문제지와 정답지(2페이지)를 자동으로 생성하여 다운로드합니다.
- **한글 폰트 완벽 지원**: `NanumGothic-Bold` 폰트를 사용하여 PDF 출력 시 한글 제목과 리스트가 깨지지 않고 선명하게 출력됩니다.
- **UI 편의 기능**: 모든 입력창을 즉시 비울 수 있는 '전체 초기화' 버튼을 제공합니다.



## 🛠 기술 스택

- **Language**: Python 3.10+
- **Framework**: Flask
- **Data Format**: JSON (words.json 기반 단어 관리)
- **Library**: ReportLab (PDF Generation)
- **Deployment**: Render
- **Frontend**: HTML5, CSS3, JavaScript (Jinja2 Template, Fetch API)
